# Fluid Music Remote

一款专为 Windows 设计的跨设备音乐远程控制器。

通过它，您可以直接在手机、平板或其他任何设备的浏览器上，实时同步并控制电脑端正在播放的音乐及系统音量。

## 核心功能

* **全平台同步**：同步显示歌名、歌手、高清专辑封面及播放进度。
* **丝滑视觉**：
  * **流体背景**：网页端背景颜色随当前播放的专辑封面动态变换，极具沉浸感。
  * **自驱动进度条**：采用线性插值算法，实现 20fps 的丝滑进度滑动。
* **深度音量控制**：支持系统全局音量调节，提供滑动条快速调节及 `+-5%` 步进微调按键。
* **广泛兼容**：完美支持 Spotify、网易云音乐、QQ音乐、YouTube (浏览器)、Chrome 媒体播放等。

## 快速开始

您可以根据自己的需求，选择以下任意一种方式运行：

### 选项 A：直接运行（推荐）
1. 前往 [Releases](https://github.com/Sakuramonth/Fluid-Music-Remote/releases/tag/windows) 页面下载最新的 `Fluid-Music-Remote.exe`。
2. 双击运行即可。

### 选项 B：源码运行

1. **安装环境**：确保电脑已安装 Python 3.10+。
2. **下载依赖**：在项目根目录打开终端，运行：

```bash
pip install -r requirements.txt
```

3. **运行程序**：执行以下命令启动控制端：

```bash
python main.py
```

## 连接方式

1. **启动程序**：运行后，观察电脑端左侧的 `SYSTEM LOG` 日志窗口。
2. **获取地址**：找到日志中显示的局域网地址（例如：`http://192.168.x.x:5000`）。
3. **同网连接**：确保手机（或平板）与电脑连接在**同一个 网络环境** 下。
4. **开始控制**：在手机浏览器中输入该地址，即可进入流体控制界面。

> **小贴士**：在手机浏览器菜单中选择 **“添加到主屏幕”**，可以隐藏浏览器地址栏，获得接近原生 App 的全屏沉浸式体验。

## 操作说明

* **播放/暂停**：点击屏幕中心的圆形大按钮。
* **切换曲目**：点击左右两侧的 `⏮` (上一曲) 或 `⏭` (下一曲) 图标。
* **音量控制**：
  * **拖动滑块**：手指左右滑动进度条，快速调节全局音量。
  * **点击 − / +**：点击滑块两端的按钮，以 **5%** 为步长进行精确调节。
